# Prediccion del Tratamiento de Gliomas Difusos con DL
En este repositorio se encuentra el código desarrollado para el Trabajo de Fin de Grado (TFG) del grado de Ingeniería Biomédica
El objetico del proyecto es la creación, entrenamiento y evaluación de redes neuronales convolucionales para la predicción ante el tratamiento de gliomas difusos en base a datos de imágen y clínicos del paciente.

# Estructura del Directorio
El directorio tiene unas ramas principales, en donde se organiza el código según su uso: Preprocesamiento, Figuras y Red-neuronal.

# Requisitos del Sistema 
Para ejecutar el código de Python no se requiere de ninguna librería externa.
Para ejecutar el código de MATLAB es necesario disponer de:
- MATLAB R2024a (o superior)
- Deep Learning Toolbox
- Image Processing Toolbox
- RAM: 16 (mínimo) o 32 (recomendado)
- Procesador: Intel Core i5 (mínimo) o Intel Core i7 o i9 (redomendado)

# Instrucciones de Uso
Simplemente, se abre el código y está listo para ser ejecutado (botón de run).

# Datos Utilizados
El dataset utilizado ha sido el de _, el cual contiene imágenes y datos clínicos de cada paciente. 
El CSV maestro contiene los datos que se utilizan ya procesados y normalizados, pero para ejecutar el código se tiene que cambiar los directorios de las imágenes utilizadas.
